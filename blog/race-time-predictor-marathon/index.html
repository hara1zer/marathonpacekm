Race predictor marathon time (combine multiple races)

Single-race predictors can swing wildly depending on whether you’re more “speed-strong” or “endurance-strong.” A simple improvement is to predict your marathon from each distance (5K/10K/half) and then combine them into a weighted estimate.

How the combined predictor works

  1. Predict marathon time from each race using the same time–distance model (Riegel-style).
  2. Weight longer races more heavily (half marathon > 10K > 5K) because they reflect endurance better.
  3. Return a range (conservative ↔ aggressive) instead of pretending there’s one perfect number.

Model background: Riegel / endurance power-law. Broader review of prediction models: MDPI review.

Why the combined approach is more realistic

Prediction accuracy improves when models use more data points and account for how individuals differ. Large-dataset work shows you can build better predictors from runners’ histories than from a single result.
